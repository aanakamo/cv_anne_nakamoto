# Anne Nakamoto's CV

Forked from [markdown-cv](https://github.com/elipapa/markdown-cv) and tweaked a bit.
- edit `index.md` to add CV contents
- edit `_config.yml` to change style (using kjhealy)
- edit files in `media` to alter font, colors, etc. (currently have edited kjhealy files to change colors and font sizes)

To make live: (from the repo) Settings -> pages -> publish site
- afterwards, new commits automatically update the site
